💰**Expense Tracker**

A simple desktop Expense & Income Tracker built using Python (Tkinter) to manage daily finances with ease.

✨ **Features**
➕ Add Income & Expenses
❌ Delete Entries
🧮 Real-time Balance Calculation
💾 Auto-save & Load Data
⚠️ Input Validation Alerts

🛠️ **Tech Stack**
Python
Tkinter (GUI)
File Handling (Local Storage)

🚀 **Getting Started**
python expense_tracker.py

Make sure Python is installed on your system.

📌 **Usage**
Enter Description and Amount
Choose:
Add Expense
Add Income
Select an item → click Delete to remove
Press Enter to quickly add an expense
View updated Total Balance instantly

📁 **Files**
expense_tracker.py   # Main application
expenses.txt         # Stored data

⚙️ **How It Works**
Income ➝ added to total
Expenses ➝ subtracted from total
Data saved automatically in expenses.txt
